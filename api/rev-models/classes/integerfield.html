Class: IntegerField

A IntegerField stores an integer (whole number) value only. Numbers with decimal values are not allowed. Use the @IntegerField() decorator to add an IntegerField to your model

Accepted Values: a number representing an integer

import { IntegerField } from 'rev-models';

class MyClass {
    @IntegerField()
        quantity: number;
}
