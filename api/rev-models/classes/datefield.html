Class: DateField

A DateField, as it's name suggests, holds just a Date value (no time value is stored). Use the @DateField() decorator to add a DateField to your model

Accepted Values: a string in the format YYYY-MM-DD

import { DateField } from 'rev-models';

class MyModel {
    @DateField()
        date_registered: string;
}
