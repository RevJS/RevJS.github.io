Class: BooleanField

A BooleanField stores a true / false value Use the @BooleanField() decorator to add a BooleanField to your model

Accepted Values: a boolean

import { BooleanField } from 'rev-models';

class MyModel {
    @BooleanField()
        is_awesome: boolean;
}
