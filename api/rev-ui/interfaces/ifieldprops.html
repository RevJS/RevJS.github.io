Interface: IFieldProps

The <Field /> component is designed to be included in a <DetailView />. When passed the name of a model field, it will render an appropriate input field to allow the value of the field to be modified.

If any validation errors occur, a <Field /> will display the corresponding validation message(s) and allow the error to be corrected.
